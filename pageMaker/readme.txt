PageMaker.exe для cms "s1".
PageMaker.exe разбивает один большой файл (например книгу), на страницы по заданному количеству символов. 


1. Создаем каталог в s1, через панель управления.
2. Далее заходим в папку datebase, находим "файл меню", первая буква m, а вторая номер. Пример: m1, m2. Находим нужный, по имени внутри.
3. Создаем пустую директорию и перемещаем туда PageMaker.exe и файл книги/документа books.txt
4. Запускаем PageMaker
5. Копируем лимит страниц(10000-19999 цифры могут быть разные в файле m. Свой диапазон вносить нельзя!) для каталога из "m" в PageMaker
6. Вводим название файла. books.txt
7. Вводим количество символов 
8. y/n
9. Файлы сгенерированы inf(с технической информацией) и p (сам текст)
10. В окне будет написано сколько файлов создано и страниц, последние запоминаем

11. Открываем "Файл меню" из пункта 2, вносим номер
	[0] dir
	[1] 10000-19999
	[2] Книга
	[3] 1
	[4] сюда пишем количество страниц
	[5] и сюда пишем количество страниц
	[6] 0

12. Копируем файлы из директории в папку database
